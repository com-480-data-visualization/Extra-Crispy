# Milestone 1 – Data Cleaning & Analysis

## Environment Setup

### 1. Create a Conda Virtual Environment

```bash
conda create -n extra-crispy python=3.12 -y
```

### 2. Activate the Environment

```bash
conda activate extra-crispy
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

## Running the Notebook

After activating the environment, launch Jupyter and open the cleaning notebook:

```bash
jupyter notebook data_clean.ipynb
```
